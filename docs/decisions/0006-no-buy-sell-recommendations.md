# ADR 0006: 매수/매도 추천 금지

- 날짜: 2026-05-15
- 상태: 채택됨 (Accepted)
- 결정자: 병승 (4월 노션에서 팀 합의됨)

## 컨텍스트

한국 자본시장법상 금융투자업자 등록 없이 투자자에게 종목 매수/매도 권유를 하는 것은 불법. 학습 프로젝트라도 컨퍼런스 시연 시 외부 노출 가능성 있음. 또한 LLM이 무의식적으로 "삼성전자 매수 추천" 같은 문장 생성할 수 있음.

## 결정

1. **모든 사용자 노출 출력에서 "매수/매도" 직접 추천 표현 금지.**
2. 허용 표현: "이 종목은 X 특성이 있습니다", "Y 섹터 비중이 높습니다", "Z 리스크에 노출됩니다"
3. 금지 표현: "이 종목 매수 권장", "이 종목 매도 추천", "buy this", "sell this"
4. **LLM 출력 함수는 반드시 `shared.disclaimers.attach()` 호출.**
5. PR 리뷰 시 grep으로 자동 체크 (`safe-pr` 스킬).
6. CI에서도 lint 룰로 추가 검토.

## 결과

- 좋은 점:
  - 법적 리스크 차단.
  - "정보 제공" 톤이 오히려 학습 도구 컨셉에 적합.
- 트레이드오프: LLM 프롬프트가 길어짐 (가드레일 명시). → 시스템 프롬프트에 한 줄 추가로 해결.

## 대안

- 면책만 첨부하고 매수/매도 표현 허용: 면책이 만능 면죄부 아님. 기각.
- 사용자가 명시적으로 "추천해줘" 했을 때만 허용: 회피 어려움. 기각.
